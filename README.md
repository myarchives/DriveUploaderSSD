# DriveUploader OATH2.0 APP

## Installation

### Package Manager, [npm](https://www.npmjs.com)

```bash
$ npm install
```

### Developmnet Environment

```bash
$ npm start
# run the application stage

$ npm dev
# run the application with nodeman for development purposes

```
